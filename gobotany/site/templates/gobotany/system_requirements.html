System Requirements

Go Botany runs on a wide range of computers, from desktops and laptops to tablets and smartphones.

Desktops, Laptops

Supported are the current and recent versions of the following browsers: Chrome, Edge, Firefox, and Safari.

Some earlier versions of the browsers listed, as well as late versions of Internet Explorer (10, 11), should also mostly work but are no longer supported.

Tablets, Smartphones

Supported are standard and mainstream browsers on the current and previous major versions of iOS and Android. These include Safari, Firefox, Chrome, and Chrome-based stock Android browsers.

Devices that run these include: iPad, iPhone, iPod Touch, Samsung Galaxy, Amazon Fire, Google Pixel, and others.

Feedback

If you experience trouble using a system that appears to be supported, please tell us about it. Please mention the device make and model, operating system version, and browser.
