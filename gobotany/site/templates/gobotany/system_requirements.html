System Requirements

Go Botany runs on a wide range of computers, from desktops and laptops to tablets and smartphones.

Desktops, Laptops

Supported are the current and previous major versions of the following browsers: Chrome, Internet Explorer, Safari, Firefox, and Opera.

Some earlier versions of the browsers listed should also work. Microsoft Edge should work but is not officially supported yet.

Tablets, Smartphones

Supported are the standard browsers on the current and previous major versions of iOS and Android.

Devices that run these include: iPad, iPhone, iPod touch, Samsung Galaxy, Nexus, and others.

Feedback

If you experience trouble using a system that appears to be supported, please write and tell us about it. State the device make and model, operating system version, and browser.
